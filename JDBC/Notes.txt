SQL is a Declarative language
JAVA is a Imparative language.

SQL->
DDL- Data Defination Language(creating the table, dropping the table)
DML- data manipulation language (want to chnage the data in the table , insert,delete, update)
DCL- data control language(permission on data, grant, revoke
TCL- transaction control language(commit,rollback) in RDBMS data will be atomic(a single truth)
DQL- data query language(name of a person who got the highest marks)
